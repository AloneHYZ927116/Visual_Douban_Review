## 3、程序结构

* **main.py**
  * 功能：运行程序的***切入点***（项目的主程序）
  * 组织
    * 通过用户输入获取电影全部评论http地址
    * 创建一个电影评论实例
    * 创建一个电影评论可视化实例
    * 可视化电影评论，以帮助用户了解电影，选择电影
* **Reviews.py**
  * 内容：一个关于豆瓣电影评论的***类***
  * 组织
* **get_reviews.py**
  * 内容：一个定量获取豆瓣电影评论的***函数***
  * 组织
